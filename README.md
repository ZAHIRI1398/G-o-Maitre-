# ZÈOMÈTRIE

Application web de géométrie interactive pour l'enseignement des mathématiques.

## Fonctionnalités

- **Outils de base**
  - Points
  - Segments
  - Droites
  - Cercles
  - Polygones avec remplissage en couleur
  - Texte

- **Outils géométriques**
  - Milieu d'un segment
  - Mesure d'angles
  - Droites parallèles
  - Droites perpendiculaires

- **Fonctionnalités avancées**
  - Import/Export PDF
  - Multi-pages
  - Gomme intelligente
  - Choix des couleurs

## Utilisation

1. Ouvrez l'application dans votre navigateur
2. Sélectionnez un outil dans la barre d'outils
3. Cliquez sur le canvas pour dessiner
4. Utilisez la gomme pour effacer
5. Importez ou exportez en PDF selon vos besoins

## Compatibilité

L'application fonctionne sur :
- Windows
- MacOS
- Tablettes (iOS/Android)
- Tous les navigateurs modernes
